第1章：常用公式讲解

掌握WPS表格中最实用的函数

SUM函数 - 求和计算

=SUM(数值1, [数值2], ...)

参数说明：

数值1 要求和的第一个数值或单元格区域
数值2 （可选）其他要相加的数值或区域
💡 实现原理：将指定范围内的所有数值相加，返回总和。支持最多255个参数，可以是数字、单元格引用或区域。
A
1成绩
285
392
478
596
=SUM(A2:A5)
↓
85+92+78+96
↓
= 351

AVERAGE函数 - 平均值计算

=AVERAGE(数值1, [数值2], ...)

参数说明：

数值1 要计算平均值的第一个数值或单元格区域
数值2 （可选）其他要计算平均值的数值或区域
💡 实现原理：先计算所有数值的总和，再除以数值的个数。自动忽略空单元格和文本。

=AVERAGE(A2:A6)
↓
总和：80+90+85+95 = 438
个数：4个
↓
= 87.6

MAX/MIN函数 - 最大最小值

=MAX(数值1, [数值2], ...)
=MIN(数值1, [数值2], ...)

参数说明：

数值1 要查找最大值/最小值的第一个数值或单元格区域
数值2 （可选）其他要比较的数值或区域
💡 实现原理：遍历所有数值进行比较，MAX返回最大值，MIN返回最小值。支持最多255个参数。

IF函数 - 逻辑判断

=IF(条件, 真值, [假值])

参数说明：

条件 要判断的逻辑表达式，结果为TRUE或FALSE
真值 条件为TRUE时返回的值
假值 （可选）条件为FALSE时返回的值
💡 实现原理：先计算条件表达式的值，如果为TRUE则返回真值，为FALSE则返回假值。支持嵌套使用实现多条件判断。
AB
1成绩结果
285?
=IF(A2>=60,"及格","不及格")
↓
判断：85 >= 60 ?
TRUE ✓
↓
= "及格"

VLOOKUP函数 - 垂直查找

=VLOOKUP(查找值, 区域, 列号, [匹配方式])

参数说明：

查找值 要在首列查找的值
区域 包含查找值和返回值的数据区域
列号 返回值所在区域的列序号
匹配方式 FALSE=精确匹配，TRUE=近似匹配
💡 实现原理：在区域的首列查找匹配值，找到后返回该行指定列的数据。适用于数据表查询、信息匹配等场景。
AB
1编号姓名
2A001张三
3A002李四
4A003王五
=VLOOKUP("A002",A:B,2,FALSE)
↓
1. 在A列查找"A002"
2. 找到第3行
3. 返回B列值
↓
= "李四"

第2章：数组应用

一次性处理多组数据

什么是数组公式

数组公式可以对一组或多组值执行计算，返回单个或多个结果

与普通公式的区别：

  • 普通公式：一次处理一个值
  • 数组公式：一次处理一组值
  • 可以执行复杂的批量计算

输入方法：

  1. 输入公式内容
  2. 按 Ctrl + Shift + Enter
  3. 公式自动加上大括号{}
示例：{=SUM(A1:A10*B1:B10)}

SUMPRODUCT 函数 - 数组乘积求和

=SUMPRODUCT(数组1, [数组2], ...)

功能说明：

  • 将多个数组的对应元素相乘
  • 再将所有乘积相加
  • 无需按Ctrl+Shift+Enter
💡 原理：数量×单价=金额，再求和
ABC
1数量单价金额
21050500
3580400
4830240
=SUMPRODUCT(A2:A4,B2:B4)
= 10×50 + 5×80 + 8×30
= 1140

TRANSPOSE 函数 - 行列转置

=TRANSPOSE(数组)

功能说明：

  • 将行数据转为列数据
  • 将列数据转为行数据
  • 输入后按Ctrl+Shift+Enter
💡 应用场景：数据透视、报表转换、横向纵向对比
原数据（横向）：
姓名语文数学
张三8590
→
转置后（纵向）：
姓名张三
语文85
数学90
{=TRANSPOSE(A1:C2)}

第3章：常用快捷键

提升操作效率的必备技能

导航与选择

快捷键功能
Ctrl + Home跳到A1单元格
Ctrl + 方向键跳到数据边界
Ctrl + Shift + 方向键选择到数据边界
Ctrl + A全选

💡 使用技巧

Ctrl + Home：无论当前在哪里，一键回到表格开头A1单元格

Ctrl + 方向键：快速跳转到连续数据的边缘，比滚动鼠标快10倍

Ctrl + Shift + 方向键：快速选中大片数据，选中后可直接复制、删除或格式化

提示：选中数据后，按Delete可清空内容，保留格式

编辑操作

快捷键功能
F2编辑单元格
F4切换引用方式
Ctrl + D向下填充
Ctrl + R向右填充

💡 使用技巧

F2：双击单元格或按F2进入编辑模式，可直接修改内容，比重新输入快

F4：在公式中切换引用方式：A1 → $A$1 → A$1 → $A1，循环切换

Ctrl + D/R：选中区域后，将第一个单元格的内容或公式填充到整个区域

提示：先输入公式，选中区域，按Ctrl+D可批量计算，比拖动填充柄更精准

基础操作

快捷键功能

Ctrl + Shift + ;输入时间

Ctrl + R向右填充
Ctrl + T创建超级表格（自动筛选、格式化、汇总）

第4章：进阶技巧

数据处理的高级方法

数据透视表

快速汇总、分析大量数据
  1. 选择数据区域
  2. 插入 → 数据透视表
  3. 拖拽字段到行/列/值区域

条件格式

根据条件自动设置单元格格式

  • 突出显示单元格规则
  • 数据条 / 色阶 / 图标集
  • 自定义公式规则

数据验证

限制单元格输入内容

常用验证类型：

  • 整数 / 小数
  • 序列（下拉列表）
  • 日期 / 时间
  • 文本长度

名称管理器

定义名称 → 简化公式引用

示例：将A1:A10定义为"销售额"

=SUM(销售额)
