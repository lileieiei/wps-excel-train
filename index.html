🔢 SUM函数 - 求和计算

=SUM(数值1, [数值2], ...)

📋 参数说明

数值1 要求和的第一个数值或单元格区域
数值2 （可选）其他要相加的数值或区域，最多255个

应用场景

计算销售总额、工资合计、库存总量等需要累加数值的场景。

💡 小贴士：可以直接选中单元格区域，如 =SUM(A1:A10)

⬇️

⬇️
💰 总销售额 = ¥42,800

📊 AVERAGE函数 - 平均值计算

=AVERAGE(数值1, [数值2], ...)

📋 参数说明

数值1 要计算平均值的第一个数值或单元格区域
数值2 （可选）其他要计算的数值或区域

应用场景

计算平均成绩、平均销售额、平均库存周转天数等。

✅ 注意：自动忽略空单元格和文本，只计算数值

3李四92

计数： 4 个数值
⬇️
📈 平均业绩 = 87.75 分

🔀 IF函数 - 条件判断

=IF(条件, 条件为真时返回值, 条件为假时返回值)

📋 参数说明

🔍 VLOOKUP函数 - 垂直查找

🔢 什么是数组公式？

一次计算，批量处理

📊 普通公式 vs 数组公式

❌ 普通公式（低效）
=A1*B1

每个单元格单独计算，需要拖拽填充

⬇️
✅ 数组公式（高效）
=A1:A10*B1:B10

一次计算整个区域，自动填充结果

✨ 数组公式优势
  • 计算速度更快
  • 公式更简洁
  • 减少出错概率
  • 内存占用更少

应用场景

  • 批量计算销售额（单价×数量）
  • 多条件加权求和
  • 复杂的数据转换

🧮 SUMPRODUCT函数

=SUMPRODUCT(数组1, 数组2, ...)

📋 功能说明

先对数组进行乘法运算，再对结果求和。支持多组数组相乘后累加。

经典应用：加权求和

计算加权平均分、加权销售额、综合评分等。

💡 公式原理：
SUMPRODUCT = Σ(数组1[i] × 数组2[i])

🎬 加权成绩计算

科目成绩权重
数学900.4
英语850.3
物理880.3
=SUMPRODUCT(B2:B4, C2:C4)
⬇️
90 × 0.4 = 36
85 × 0.3 = 25.5
88 × 0.3 = 26.4
⬇️
📊 加权平均分 = 87.9

📊 数据透视表

快速汇总分析大量数据

🎯 什么是数据透视表？

数据透视表是一种交互式的数据汇总工具，可以快速对大量数据进行分类、汇总和分析。

✨ 核心优势
  • 无需写公式，拖拽即可
  • 自动汇总计算
  • 支持多维度分析
  • 数据更新后一键刷新

📝 创建步骤

  1. 选中数据区域
  2. 点击 插入 → 数据透视表
  3. 选择放置位置
  4. 拖拽字段到行/列/值区域

应用场景

销售分析、财务报表、库存统计、人事数据分析等

✅ 数据验证

防止输入错误，保证数据质量

🎯 功能介绍

数据验证可以限制单元格的输入内容，防止错误数据录入。

✓ 限制输入范围（如1-100）
✓ 创建下拉列表
✓ 设置日期范围
✓ 自定义验证规则

📝 设置方法

  1. 选中要限制的单元格
  2. 点击 数据 → 有效性
  3. 选择允许的条件类型
  4. 设置具体规则

典型应用

性别只能选男/女、日期不能早于今天、金额必须是正数等

📅 DATEDIF函数 - 计算工龄和年龄

=DATEDIF(开始日期,结束日期,"单位")

📋 参数说明

"Y" 返回完整年数（计算年龄、工龄）
"M" 返回完整月数
"D" 返回天数

HR实战应用

自动计算员工工龄、年龄、司龄，用于福利计算、退休预警等。

💡 注意：这个函数是隐藏函数，不会自动提示，但可以直接使用

🎬 实例演示

计算年龄：
=DATEDIF(出生日期,TODAY(),"Y")
计算工龄：
=DATEDIF(参加工作日期,TODAY(),"Y")&"年"&DATEDIF(参加工作日期,TODAY(),"YM")&"个月"

结果示例

孙家龙：工龄 40年，年龄 60岁

陈关敏：工龄 26年，年龄 44岁

🔢 文本函数 - 从身份证提取信息

=MID(文本,开始位置,提取长度)

📋 身份证结构

第1-6位 地区代码（如110101是北京市东城区）
第7-14位 出生日期（YYYYMMDD）
第17位 性别码（奇数男，偶数女）

HR应用场景

自动提取出生日期、判断性别、验证身份证有效性。

🎬 实战公式

提取出生日期：
=TEXT(MID(身份证号,7,8),"0000-00-00")
判断性别：
=IF(MOD(MID(身份证号,17,1),2)=1,"男","女")
计算年龄（从身份证）：
=DATEDIF(TEXT(MID(身份证号,7,8),"0000-00-00"),TODAY(),"Y")
✅ 验证：第2位员工张卫华，身份证110102197201070010，提取后：1972-01-07，性别男

💰 薪资计算综合应用

=基本工资+职务工资+岗位津贴

📋 计算项目

应发工资 =SUM(基本工资:岗位津贴)
个税计算 =MAX((应发工资-5000)*税率-速算扣除数,0)
实发工资 =应发工资-个税-社保

进阶：按职称加津贴

高工额外+200，工程师+100，其他+0

=SUM(工资项)+IF(职称="高工",200,IF(职称="工程师",100,0))
