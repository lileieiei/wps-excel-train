🔢 SUM函数 - 求和计算

=SUM(数值1, [数值2], ...)

📋 参数说明

数值1 要求和的第一个数值或单元格区域
数值2 （可选）其他要相加的数值或区域，最多255个

应用场景

计算销售总额、工资合计、库存总量等需要累加数值的场景。

💡 小贴士：可以直接选中单元格区域，如 =SUM(A1:A10)

🎬 动画演示

AB
1产品销售额
2产品A¥8,500
3产品B¥12,300
4产品C¥6,800
5产品D¥15,200
=SUM(B2:B5)
⬇️
8500+ 12300+ 6800+ 15200
⬇️
💰 总销售额 = ¥42,800

📊 AVERAGE函数 - 平均值计算

=AVERAGE(数值1, [数值2], ...)

📋 参数说明

数值1 要计算平均值的第一个数值或单元格区域
数值2 （可选）其他要计算的数值或区域

应用场景

计算平均成绩、平均销售额、平均库存周转天数等。

✅ 注意：自动忽略空单元格和文本，只计算数值

🎬 动画演示

AB
1员工业绩
2张三85
3李四92
4王五78
5赵六96
=AVERAGE(B2:B5)
⬇️
求和： 85 + 92 + 78 + 96 = 351
计数： 4 个数值
⬇️
📈 平均业绩 = 87.75 分

🔀 IF函数 - 条件判断

=IF(条件, 条件为真时返回值, 条件为假时返回值)

📋 参数说明

条件 要判断的逻辑表达式，结果为TRUE或FALSE
真值 条件为TRUE时返回的结果
假值 条件为FALSE时返回的结果

应用场景

成绩等级评定、销售达标判断、库存预警、费用报销审核等。

🎬 成绩等级判定

=IF(B2>=60, "及格", "不及格")
学生分数结果
张三85?
⬇️
85 >= 60 → ✓ TRUE
⬇️
🎉 结果：及格

多条件判断，实现复杂逻辑

📊 成绩等级评定系统

  IF(B2>=80, "良好",
    IF(B2>=60, "及格", "不及格")))

💡 新函数推荐：WPS/Excel 2019+ 支持 IFS 函数，可以更简洁地写多条件判断： =IFS(B2>=90,"优秀",B2>=80,"良好",B2>=60,"及格",TRUE,"不及格")

🔍 VLOOKUP函数 - 垂直查找

=VLOOKUP(查找值, 查找区域, 返回列序号, [精确匹配])

📋 参数说明

查找值 要在第一列查找的内容
查找区域 包含查找值和返回值的数据表
返回列 要返回的数据在第几列（从1开始）
精确匹配 FALSE=精确匹配，TRUE=近似匹配

应用场景

根据员工编号查姓名、根据商品编码查价格、根据身份证号查信息等。

🎬 员工信息查询

工号姓名部门工资
A001张三销售8500
A002李四技术12000
A003王五人事6800
查找：A001 的工资
=VLOOKUP("A001", A2:D4, 4, FALSE)
⬇️
1️⃣ 在第1列找 "A001" → 找到第1行
2️⃣ 返回第4列的数据 → 工资列
⬇️
💰 结果：¥8,500

🔢 什么是数组公式？

一次计算，批量处理

📊 普通公式 vs 数组公式

❌ 普通公式（低效）
=A1*B1

每个单元格单独计算，需要拖拽填充

⬇️
✅ 数组公式（高效）
=A1:A10*B1:B10

一次计算整个区域，自动填充结果

✨ 数组公式优势
  • 计算速度更快
  • 公式更简洁
  • 减少出错概率
  • 内存占用更少

应用场景

  • 批量计算销售额（单价×数量）
  • 多条件加权求和
  • 复杂的数据转换

🧮 SUMPRODUCT函数

=SUMPRODUCT(数组1, 数组2, ...)

📋 功能说明

先对数组进行乘法运算，再对结果求和。支持多组数组相乘后累加。

经典应用：加权求和

计算加权平均分、加权销售额、综合评分等。

💡 公式原理：
SUMPRODUCT = Σ(数组1[i] × 数组2[i])

🎬 加权成绩计算

科目成绩权重
数学900.4
英语850.3
物理880.3
=SUMPRODUCT(B2:B4, C2:C4)
⬇️
90 × 0.4 = 36
85 × 0.3 = 25.5
88 × 0.3 = 26.4
⬇️
📊 加权平均分 = 87.9

📊 数据透视表

快速汇总分析大量数据

🎯 什么是数据透视表？

数据透视表是一种交互式的数据汇总工具，可以快速对大量数据进行分类、汇总和分析。

✨ 核心优势
  • 无需写公式，拖拽即可
  • 自动汇总计算
  • 支持多维度分析
  • 数据更新后一键刷新

📝 创建步骤

  1. 选中数据区域
  2. 点击 插入 → 数据透视表
  3. 选择放置位置
  4. 拖拽字段到行/列/值区域

应用场景

销售分析、财务报表、库存统计、人事数据分析等

🎨 条件格式

让数据一目了然

🌈

色阶

用颜色深浅表示数值大小

📊

数据条

用条形长度直观展示

🔴

图标集

用图标表示状态

🔴 🟡 🟢
💡 实用技巧：选中数据 → 开始 → 条件格式 → 选择规则类型。可以设置大于/小于某个值时自动标红，或前10%高亮显示。

✅ 数据验证

防止输入错误，保证数据质量

🎯 功能介绍

数据验证可以限制单元格的输入内容，防止错误数据录入。

✓ 限制输入范围（如1-100）
✓ 创建下拉列表
✓ 设置日期范围
✓ 自定义验证规则

📝 设置方法

  1. 选中要限制的单元格
  2. 点击 数据 → 有效性
  3. 选择允许的条件类型
  4. 设置具体规则
  5. （可选）设置输入提示和错误警告

典型应用

性别只能选男/女、日期不能早于今天、金额必须是正数等
