# 练习1：员工工资核算

## 练习目标
掌握IF函数、VLOOKUP函数和数组公式的实际应用

## 练习时长
20分钟

## 数据说明
- 基础数据包含8名员工的基本信息
- 需要计算：全勤奖、社保扣除、公积金、个税、实发工资

## 练习步骤

### 步骤1：建立工资表结构
创建以下列：
- 员工编号、姓名、部门、基本工资、岗位津贴
- 出勤天数、应出勤天数、绩效奖金
- 全勤奖、社保扣除、公积金、个税、实发工资

### 步骤2：导入基础数据
将基础数据.csv中的数据复制到表格中

### 步骤3：计算全勤奖
使用IF函数判断：
- 如果出勤天数=应出勤天数，全勤奖=500
- 否则全勤奖=0

公式示例：
```
=IF(G2=H2, 500, 0)
```

### 步骤4：计算社保扣除
社保扣除 = 基本工资 × 社保扣除比例

公式示例：
```
=D2*0.08
```

### 步骤5：计算公积金
公积金 = 基本工资 × 公积金比例

公式示例：
```
=D2*0.12
```

### 步骤6：计算个税（简化版）
应纳税所得额 = 基本工资 + 岗位津贴 + 绩效奖金 + 全勤奖 - 社保扣除 - 公积金 - 5000

如果应纳税所得额 <= 0，个税 = 0
如果 0 < 应纳税所得额 <= 3000，个税 = 应纳税所得额 × 0.03
如果 3000 < 应纳税所得额 <= 12000，个税 = 应纳税所得额 × 0.1 - 210

使用嵌套IF函数：
```
=IF((D2+E2+J2+K2-L2-M2-5000)<=0, 0, IF((D2+E2+J2+K2-L2-M2-5000)<=3000, (D2+E2+J2+K2-L2-M2-5000)*0.03, (D2+E2+J2+K2-L2-M2-5000)*0.1-210))
```

### 步骤7：计算实发工资
实发工资 = 基本工资 + 岗位津贴 + 绩效奖金 + 全勤奖 - 社保扣除 - 公积金 - 个税

### 步骤8：制作工资条
为每位员工生成独立的工资条格式

### 步骤9：部门汇总
使用数据透视表或SUMIF函数统计各部门的工资总额

## 参考答案
完成后的表格应包含：
- 所有员工的完整工资信息
- 正确的公式计算结果
- 格式化的工资条
- 部门汇总数据

## 提示
1. 使用绝对引用($A$1)和相对引用(A1)时注意区分
2. 复杂的嵌套IF可以使用IFS函数（WPS新版本支持）
3. 可以使用条件格式高亮显示异常数据
