# 练习2：销售数据统计分析

## 练习目标
掌握数据透视表、SUMIF/COUNTIF函数和图表制作

## 练习时长
15分钟

## 练习步骤

### 步骤1：导入数据
将销售数据.csv导入到WPS表格

### 步骤2：计算销售额
添加"销售额"列，公式：=数量*单价

### 步骤3：使用SUMIF统计
1. 按销售员统计总销售额
2. 按区域统计总销售额
3. 按产品统计总销售额

公式示例：
```
=SUMIF(B:B, "张三", G:G)
```

### 步骤4：使用COUNTIF统计
1. 统计每个销售员的订单数
2. 统计每个区域的订单数

公式示例：
```
=COUNTIF(B:B, "张三")
```

### 步骤5：创建数据透视表
1. 选择数据区域
2. 插入 → 数据透视表
3. 行：销售员，值：销售额（求和）
4. 复制透视表，改为按区域分析

### 步骤6：制作图表
1. 选择透视表数据
2. 插入 → 图表 → 柱状图
3. 添加图表标题和数据标签

### 步骤7：条件格式
对销售额列应用条件格式：
- 数据条显示大小
- 色阶显示高低

## 扩展练习
1. 计算每个销售员的平均订单金额
2. 找出销售额最高的日期
3. 制作饼图显示各产品占比
