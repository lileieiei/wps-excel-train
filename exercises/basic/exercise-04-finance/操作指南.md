# 练习4：个人理财记账表

## 练习目标
掌握数据验证、SUMIF函数和图表制作

## 练习时长
15分钟

## 练习步骤

### 步骤1：建立收支记录表
创建包含以下列的表格：
- 日期、类别、项目、收入、支出、余额

### 步骤2：设置数据验证
对"类别"列设置下拉列表：
1. 选择类别列
2. 数据 → 数据验证 → 序列
3. 来源：餐饮,交通,购物,娱乐,工资,其他

### 步骤3：计算余额
余额 = 上期余额 + 本期收入 - 本期支出

第一行公式：
```
=D2-E2
```

第二行起公式：
```
=F2+D3-E3
```

### 步骤4：按类别汇总
创建汇总表，使用SUMIF统计各类别的收入和支出

公式示例：
```
=SUMIF(收支记录!B:B, "餐饮", 收支记录!D:D)
```

### 步骤5：计算月度总计
总收入 = SUM(收入列)
总支出 = SUM(支出列)
结余 = 总收入 - 总支出

### 步骤6：制作图表
1. 收入支出对比柱状图
2. 各类别支出占比饼图
3. 每日余额趋势折线图

### 步骤7：条件格式
对支出列应用条件格式：
- 支出 > 100：红色显示
- 支出 <= 50：绿色显示
