# 练习3：库存管理表

## 练习目标
掌握SUMIF、VLOOKUP、条件格式的综合应用

## 练习时长
15分钟

## 练习步骤

### 步骤1：建立库存表
创建包含以下列的表格：
- 商品编号、商品名称、类别、单价、期初库存
- 入库数量、出库数量、当前库存、库存金额

### 步骤2：导入基础数据
从商品信息.csv导入商品基本信息

### 步骤3：计算入库数量
使用SUMIF函数，根据商品编号统计入库总数

公式示例：
```
=SUMIF(出入库记录!B:B, A2, 出入库记录!D:D)
```
注意：需要先筛选"入库"类型的记录

更精确的公式：
```
=SUMIFS(出入库记录!D:D, 出入库记录!B:B, A2, 出入库记录!C:C, "入库")
```

### 步骤4：计算出库数量
类似步骤3，统计"出库"类型的数量

### 步骤5：计算当前库存
当前库存 = 期初库存 + 入库数量 - 出库数量

### 步骤6：计算库存金额
库存金额 = 当前库存 × 单价

### 步骤7：使用VLOOKUP查询
在出入库记录表中，使用VLOOKUP查询商品名称

公式示例：
```
=VLOOKUP(B2, 商品信息!A:B, 2, FALSE)
```

### 步骤8：设置条件格式
1. 当前库存 < 50：红色背景（库存不足）
2. 当前库存 > 200：黄色背景（库存积压）
3. 使用数据条显示库存金额大小

### 步骤9：库存预警
添加"状态"列，使用IF函数：
```
=IF(H2<50, "库存不足", IF(H2>200, "库存积压", "正常"))
```
