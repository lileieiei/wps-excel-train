## 🎯 练习任务

### 任务一：计算工龄和年龄

#### 1.1 计算年龄
在空白列输入公式，根据出生日期计算年龄：

```excel
=DATEDIF(出生日期单元格, TODAY(), "Y")
```

**示例**：
```excel
=DATEDIF(H2, TODAY(), "Y")
```

#### 1.2 计算工龄
根据参加工作时间计算工龄：

```excel
=DATEDIF(参加工作时间单元格, TODAY(), "Y")
```

#### 1.3 格式化显示工龄（带年月）
```excel
=DATEDIF(G2,TODAY(),"Y")&"年"&DATEDIF(G2,TODAY(),"YM")&"个月"
```

💡 **提示**：DATEDIF是隐藏函数，输入时不会自动提示，但可以直接使用。

---

### 任务二：从身份证提取信息

#### 2.1 提取出生日期
使用MID函数从身份证第7位开始提取8位出生日期：

```excel
=TEXT(MID(I2,7,8),"0000-00-00")
```

#### 2.2 判断性别
身份证第17位为奇数是男性，偶数是女性：

```excel
=IF(MOD(MID(I2,17,1),2)=1,"男","女")
```

**公式解析**：
- `MID(I2,17,1)`：提取第17位数字
- `MOD(...,2)`：取除以2的余数
- `IF(...=1,"男","女")`：余数为1是男，否则是女

#### 2.3 计算年龄（从身份证）
```excel
=DATEDIF(TEXT(MID(I2,7,8),"0000-00-00"),TODAY(),"Y")
```

### 任务三：薪资计算

#### 3.1 计算应发工资
应发工资 = 基本工资 + 职务工资 + 岗位津贴

```excel
=SUM(J2:L2)
```
或
```excel
=J2+K2+L2
```

#### 3.2 按职称增加津贴（进阶）
高工额外+200元，工程师+100元，其他+0元：

```excel
=SUM(J2:L2)+IF(F2="高工",200,IF(F2="工程师",100,0))
```

### 任务四：多条件统计

#### 4.1 统计各部门人数
```excel
=COUNTIFS(C:C,"A部门")
=COUNTIFS(C:C,"B部门")
=COUNTIFS(C:C,"C部门")
```

#### 4.2 统计各部门男女比例
```excel
=COUNTIFS(C:C,"A部门",D:D,"男")
=COUNTIFS(C:C,"A部门",D:D,"女")
```

#### 4.3 统计各职称人数
```excel
=COUNTIFS(F:F,"高工")
=COUNTIFS(F:F,"工程师")
=COUNTIFS(F:F,"技师")
=COUNTIFS(F:F,"技术员")
```

#### 4.4 统计特定条件人数（复合条件）
统计A部门中工龄超过30年的高工人数：

```excel
=COUNTIFS(C:C,"A部门",F:F,"高工",工龄列,">30")
```

#### 4.5 计算各部门平均工资
```excel
=AVERAGEIFS(应发列,C:C,"A部门")
```

#### 4.6 计算各职称总工资
```excel
=SUMIFS(应发列,F:F,"高工")
```

### 任务六：VLOOKUP信息查询

创建一个查询区域，根据姓名查询员工信息：

#### 6.1 基础查询
假设在M1单元格输入要查询的姓名：

查询部门：
```excel
=VLOOKUP(M1,A:L,3,FALSE)
```

查询应发工资（假设在J列）：
```excel
=VLOOKUP(M1,A:L,10,FALSE)
```

#### 6.2 创建查询面板
创建一个美观的查询面板：

| 输入姓名 | [输入框] | 
|---------|---------|
| 部门 | =VLOOKUP(输入框,A:L,3,FALSE) |
| 职务 | =VLOOKUP(输入框,A:L,5,FALSE) |
| 职称 | =VLOOKUP(输入框,A:L,6,FALSE) |
| 应发工资 | =VLOOKUP(输入框,A:L,10,FALSE) |
| 工龄 | =VLOOKUP(输入框,A:L,工龄列,FALSE) |

⚠️ **注意**：确保查询列（姓名）在第一列，且无重复值。

### 任务七：条件格式高阶应用

#### 7.1 退休预警（标红）
男性≥58岁，女性≥53岁标红：

1. 选中年龄列
2. 点击 **开始** → **条件格式** → **新建规则**
3. 选择"使用公式确定要设置格式的单元格"
4. 输入公式：
```excel
=OR(AND(D2="男",年龄列>=58),AND(D2="女",年龄列>=53))
```
5. 设置格式：红色填充

#### 7.2 薪资异常高亮
工资超过平均值20%高亮显示：

```excel
=应发列>AVERAGE($M:$M)*1.2
```

#### 7.3 工龄分层（色阶）
1. 选中工龄列
2. 点击 **开始** → **条件格式** → **色阶**
3. 选择绿-黄-红色阶
