# 练习7：自动化报表生成

## 练习目标
掌握名称管理器、INDIRECT函数和模板设计

## 练习时长
30分钟

## 练习步骤

### 步骤1：创建数据源表
导入报表数据.csv，命名为"数据源"

### 步骤2：定义名称
使用名称管理器定义以下名称：

1. 月份 =OFFSET(数据源!$A$2,0,0,COUNTA(数据源!$A:$A)-1,1)
2. 销售额 =OFFSET(数据源!$B$2,0,0,COUNTA(数据源!$A:$A)-1,1)
3. 成本 =OFFSET(数据源!$C$2,0,0,COUNTA(数据源!$A:$A)-1,1)
4. 费用 =OFFSET(数据源!$D$2,0,0,COUNTA(数据源!$A:$A)-1,1)

### 步骤3：创建动态报表模板
制作报表框架：
- 标题区域
- 数据展示区域
- 图表区域

### 步骤4：使用INDIRECT动态引用
创建下拉选择器，选择要查看的月份范围：

公式示例：
```
=INDIRECT("数据源!B" & 开始行 & ":B" & 结束行)
```

### 步骤5：制作动态图表
1. 创建图表
2. 图表数据源使用名称
3. 图表随数据更新自动调整

### 步骤6：添加计算指标
计算以下指标：
- 毛利 = 销售额 - 成本
- 净利润 = 毛利 - 费用
- 毛利率 = 毛利 / 销售额
- 净利率 = 净利润 / 销售额

使用数组公式批量计算：
```
{=销售额-成本}
```

### 步骤7：设置数据验证
创建参数输入区域：
- 起始月份（下拉列表）
- 结束月份（下拉列表）
- 指标类型（下拉列表：销售额/成本/费用/毛利/净利润）

### 步骤8：制作自动汇总表
使用SUMPRODUCT根据选择的月份范围汇总：

公式示例：
```
=SUMPRODUCT((月份>=起始月份)*(月份<=结束月份)*销售额)
```

### 步骤9：添加条件格式
根据指标类型自动变色：
- 利润率 > 20%：绿色
- 利润率 10-20%：黄色
- 利润率 < 10%：红色

### 步骤10：保护模板
1. 锁定公式单元格
2. 允许编辑参数区域
3. 设置工作表保护密码

### 步骤11：测试自动化
1. 添加新的月份数据
2. 验证报表自动更新
3. 测试各种参数组合
