# 练习5：动态销售数据仪表板

## 练习目标
掌握INDEX+MATCH、动态图表、切片器的综合应用

## 练习时长
25分钟

## 练习步骤

### 步骤1：数据准备
导入销售数据.csv，确保数据格式正确

### 步骤2：创建数据透视表
1. 选择数据区域
2. 插入 → 数据透视表
3. 行：月份、区域
4. 列：产品
5. 值：销售额（求和）

### 步骤3：添加切片器
1. 选中透视表
2. 分析 → 插入切片器
3. 选择：区域、销售员
4. 调整切片器位置和大小

### 步骤4：创建动态汇总表
使用INDEX+MATCH实现动态查询：

创建查询区域：
- 下拉选择：月份、区域、产品

公式示例：
```
=INDEX(销售额区域, MATCH(选择月份, 月份列, 0), MATCH(选择产品, 产品行, 0))
```

### 步骤5：制作动态图表
1. 根据动态汇总表创建图表
2. 使用OFFSET定义动态数据区域
3. 图表随筛选条件自动更新

### 步骤6：添加条件格式
1. 目标完成率 >= 1：绿色
2. 目标完成率 < 0.9：红色
3. 使用数据条显示销售额大小

### 步骤7：美化仪表板
1. 隐藏网格线
2. 统一配色方案
3. 添加标题和说明
4. 调整布局使其美观

## 扩展练习

3. 创建趋势预测图表
