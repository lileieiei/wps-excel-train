# 练习6：财务数据多维度分析

## 练习目标
掌握SUMPRODUCT、复杂数组公式和多维度分析

## 练习时长
25分钟

## 练习步骤

### 步骤1：数据导入
导入财务数据.csv

### 步骤2：计算季度合计
使用数组公式或SUM函数计算每行的1-3月合计

公式：
```
=SUM(C2:E2)
```

### 步骤3：使用SUMPRODUCT多条件汇总
1. 按部门汇总费用
2. 按项目类型汇总费用
3. 计算预算执行率

按部门汇总示例：
```
=SUMPRODUCT((部门列="销售部")*(金额列))
```

多条件汇总示例：
```
=SUMPRODUCT((部门列="销售部")*(项目列="差旅费")*(金额列))
```

### 步骤4：计算预算执行率
预算执行率 = 实际发生 / 预算

使用数组公式计算各部门执行率：
```
{=SUM(IF(部门列="销售部", 实际列, 0)) / SUM(IF(部门列="销售部", 预算列, 0))}
```

### 步骤5：创建交叉分析表
制作部门×项目的费用矩阵：
- 行：部门
- 列：项目
- 值：SUMPRODUCT计算

### 步骤6：异常数据识别
使用条件格式：
1. 执行率 > 100%：红色预警
2. 执行率 < 80%：黄色提醒
3. 使用色阶显示费用高低

### 步骤7：制作分析图表
1. 各部门费用占比饼图
2. 各项目趋势折线图
3. 预算vs实际对比柱状图

### 步骤8：添加数据验证
创建查询界面：
- 下拉选择部门
- 动态显示该部门所有项目费用

使用INDEX+SMALL+IF数组公式提取数据
